# image-keeper

A web app for storing, deleting and labeling your images.

## Getting started

### Requirements

This project requires Node.js and Redis running.

### Steps

Clone the repository:

```
git clone https://github.com/iffypixy/image-keeper
cd ./image-keeper
```

Install dependencies and run:

```
pnpm install

pnpm run dev
pnpm run start:dev
```
